# csv_viewer
CSV files viewer

command line usage: 

    python csv_viewer.py [-h] -p PATH -s SEPARATOR

    optional arguments:
        -h, --help    show this help message and exit
        -p PATH       Path to CSV file
        -s SEPARATOR  Separator: comma, semicolon or tab
